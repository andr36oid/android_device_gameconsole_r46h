DO NOT DO THIS! USE ANYTHING BUT GAPPS.
This is heavily discouraged to install, as 1GB of RAM is quite low for GAPPS, and they will most likely crash over and over.
However if for some reason you really want to install GAPPS and slow down your system (even get annoyed by gapps notification spam), you can (but should NOT) install them like this:

Download MindTheGapps-11.0.0-arm64-20230922_081122.zip and extract it into this folder. It will be taken care of on first boot. Note, do not put it in a subfolder. It should be set up like this.

└── gapps
    ├── gapps.txt
    ├── META_INF
    ├── system
    └── toybox
